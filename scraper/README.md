1. Backend & Скрапер

Воркер периодически собирает объявления с carsensor.net. Обязательные поля: марка, модель, год, цена, цвет, ссылка. Логика upsert: обновлять существующие записи при изменениях, добавлять новые. Базовая retry-логика при сетевых ошибках.

БД PostgreSQL/MySQL. Миграции через Alembic, сидинг с дефолтным администратором. Эндпоинты:
- POST /api/login — авторизация, возврат JWT
- GET /api/cars — защищённый список автомобилей
